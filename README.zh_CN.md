# VUE3-TEMPLATE

[English](https://github.com/daixu-cn/VUE3-TEMPLATE/blob/main/README.md) | 简体中文

VUE3初始化工程模版, 默认配置: Vue、Vite、TypeScript、SCSS、Pinia、ESLint、Prettier、Husky、Element Plus、自定义/深色主题、国际化、旧版浏览器支持

## 项目指令

```shell
# 安装依赖 npm
npm install

# 安装依赖 pnpm
pnpm install

# 启动项目
npm run start

# 打包项目
npm run build

# 格式化所有文件代码格式
npm run prettier

# 提交代码
git add .
npm run commit
```
